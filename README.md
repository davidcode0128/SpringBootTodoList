# SpringBootTodoList
三層式架構實作Todo List

## 大綱

本專案利用Spring Boot實作了三層式 ( 介面層、業務邏輯層、資料訪問層 ) 架構，希望能夠藉由實作本專案，更加了解Spring Boot的運行機制。

## 做了什麼

1. 使用MySQL作為資料庫，並使用JPA作為查詢語言，實現基本的CRUD功能。
2. 使用JUnit5實作單元測試。

## 環境
+ 作業系統 ： Linux Ubuntu 20.04
+ 開發與運行環境 : JDK-17
+ 資料庫 : MySQL 8.0
+ Spring Boot版本 : 2.7.3
